govdat by SChamberlain

Interface to APIs for data on what our elected representatives are up to.

govdat is an interface to the Sunlight Labs APIs and the New York Times Congress API.

Dependencies

RJSONIO, RCurl (>= 1.6), XML, stringr, plyr

Install

install.packages("devtools") require(devtools) install_github("govdat", "SChamberlain") require(govdat)
